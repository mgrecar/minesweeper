# Overview

A simple minesweeper clone.  Basically just a personal project to learn to use Phaser.

## Prerequisites
1. Node.js and NPM

## Directions
1. `git clone git@github.com:mgrecar/minesweeper.git`
2. `cd minesweeper`
3. `npm install`
4. `npm start`
5. Navigate a web browser to `localhost:3000`
